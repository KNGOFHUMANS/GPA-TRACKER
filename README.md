GPA Tracker

This is a Java project that helps students track their GPA. It has a login system, can send emails, and also lets you sign in with Google.

Features

Login UI – Users can make an account, log in, and log out.

Google Sign-In – You can log in with your Google account.

Email Support – The app can send emails (like account updates or recovery emails).

GPA Tracking – Add classes and grades, and it will calculate your GPA automatically.

Dark Mode – Switch between light and dark themes.

How It Works

The login system checks your account info before letting you in.

Google Sign-In uses Google’s OAuth to connect your account.

The email system uses SMTP (like Gmail’s mail server) to send messages.

GPA Tracker saves your classes and grades in JSON files and shows your GPA.

What You Need

Java (JDK 17 or higher)

Gson library (for saving data in JSON)

JavaMail API (for sending emails)

Google OAuth libraries (for Google Sign-In)
